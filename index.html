Ecommerce Front-end

Nesse projeto foi desenvolvido um site de e-commerce que mostra os produtos listados no Mercado Livre.
Para isso foi utilizada a API do Mercado Livre que fornece os dados de cada um dos produtos e o código escrito lê essas informações e cria um componente para cada produto de maneira automática.
Além disso, para ter uma melhor taxa de conversão foi utilizado um recurso chamado Local Storage, que permite que os produtos adicionados ao carrinho continuem nele quando o site é carregado novamente no mesmo dispositivo.

MAfin - Michelle Abritta

Esse site foi criado pensando em performance e conversão.
Sua estrutura é simples e direta, com uma usabilidade fluida e telas totalmente adaptáveis de acordo com o dispositivo.
Mas o maior destaque para esse projeto não está na simplicidade ofertada para o usuário, mas sim na sua integração com as maiores plataformas de marketing digital disponíveis no mercado.
Ao acessar a página, o site capta as informações do usuário (seguindo a Lei Geral de Proteção de Dados) e envia para o Google e para a Meta (Facebook e Instagram), o que permite que sejam criados anúncios em ambas plataformas segmentando para visitantes do site.

Star Wars Planets Data

Nesse projeto foi desenvolvido uma tabela interativa com informações públicas a respeito dos planetas da série de filmes Star Wars.
As informações presentes na página são consultadas assim que o usuário acessa a página, por isso aparece temporariamente uma animação de carregamento.
Além disso, o usuário consegue filtrar e ordenar os dados da maneira como desejar, o que torna a consulta de informações simples, mas funcional.
As tecnologias desenvolvidas aqui podem contribuir com projetos futuros que envolvem a interação do usuário com a gestão de muitos dados, por exemplo permitir que se crie uma dieta personalizada usando filtros de alimentos que o cliente gosta e ordenando as receitas de acordo com suas calorias.
